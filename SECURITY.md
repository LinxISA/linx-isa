# Security Policy

## Reporting a vulnerability

Please use GitHub’s **“Report a vulnerability”** feature for this repository.
This routes reports to maintainers privately.

If you believe you have found a critical vulnerability that could impact users
of LinxISA components (toolchain, emulator, kernel, libc), please include:

- A clear description of the issue and impact
- Steps to reproduce (or a minimal PoC)
- Affected component(s) and versions/commits

## Supported versions

Security fixes are applied to the default branch (`main`). Releases may be cut
as needed.
